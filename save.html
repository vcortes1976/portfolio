$AVE

UI | Responsive Web Design | Brand Development

DISCOVER

$ave is a responsive web app that helps you save money quickly and manage your budget easily, from anywhere at any time. Users can view their finances at a glance, track their incomings and outgoings and create a budget profile so that the financial advice they receive is tailored to their personal goals.

My role:
Usability testing and analysis
Product design
Brand and style guide

Project duration: Approximately 8 weeks.

DEFINE

Who - Users are not new to technology but they are less familiar and comfortable with the world of finance.

What - A responsive web app allowing all data on income and expenses to be recorded easily.

When - Before a planned expense, users will input their relevant financial data into the app in order to receive information on how much they can save and for how long.

Where - The tool can be used from anywhere on any device.

Why - Saving money can be really hard and not everyone can afford to have a financial advisor.

DEVELOP

User flow diagram - I created user flows which would enable users to achieve the following:

  • As a new user, I need to be able to create a profile, so that the financial advice I receive is personalized to me, my spending habits, and tailored to my goals.
  • As a user, I want to see a dashboard of my finances clearly and visually, so that I can see how much I am spending on what at a glance.
  • As a money-saver, I need to be able to input information on the money I am receiving and spending (and on what), so that I can see an overview of my finances.
  • As a user, I want to be notified and rewarded when I have reached certain milestones throughout the saving period, so that I know whether Iâ€™m on track to reach my goal.

Wireframes - Using the Crazy 8 technique, I sketched low fidelity wireframes for all screens and created mid to high fidelity wireframes for prototyping and testing.

User tests - A number of usability issues came up which I was able to solve early in the process. I was also able to get feedback on the overall brand using mid to high fidelity prototypes.

A/B preference test - 65% preferred the brand and logo $ave.
